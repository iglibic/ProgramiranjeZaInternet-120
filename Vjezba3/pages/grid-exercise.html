This is a grid exercise

To complete this task make changes only in the /styles/grid.css file to align items according to the given task.

To setup the exercise set the display property of the grid container to show a grid with 3 rows and 3 columns which each take equal fractions of the grid.

All elements should be centered within their grid areas(cells), vertically and horizontally.
