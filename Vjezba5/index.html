Exercise 5

  1. What is the difference between let and const keywords?

    Answer: Varijable deklarirane s let imaju blok-dohvat, što znači da su dostupne samo unutar bloka u kojem su definirane. Varijable deklarirane s const također imaju blok-dohvat, ali im se vrijednost ne može ponovno dodijeliti.

  2. What is JavaScript scope, what are the different types of JavaScript scope?

    Answer: Scope određuje gdje se varijable mogu koristiti. Postoje globalni i lokalni (funkcijski i blok) dohvat.

  3. Using the steps for loose equality comparison, explain what and why will be logged to console in the following code.

    1. console.log({} == [])

      Answer: Netočno. {} i [] su objekti, ali imaju različite reference.

    2. console.log(null == undefined)

      Answer: Točno. Null i undefined jednaki su jedan drugome.

    3. console.log({} == {})

      Answer: Netočno. Riječ je o različitim referencama objekata.

    4. console.log([1] == 1)

      Answer: Točno. Niz [1] se pretvara u broj 1.

    5. console.log("1" == 1)

      Answer: Točno. "1" se pretvara u broj 1.

    6. console.log(true == 1)

      Answer: Točno. Vrijednost true pretvara se u broj 1.

    7. console.log([] == "")

      Answer: Točno. Prazan niz [] pretvara se u prazan string "".

  4. If we use strict equality operator instead of loose equality operator for the examples from the previous task, what will be different?

    Answer: Stroga jednakost (===) ne vrši konverziju tipova. Tip i vrijednost moraju se podudarati.

  5. Why does '' == false return true but '' === false return false?

    Answer: '' == false → istinito, jer se oba pretvaraju u 0, '' === false → netočno, jer su različitih tipova.

  6. In JavaScript, what are truthy/falsy values? Name them.

    Answer: Falsy vrijednosti: false, 0, '', null, undefined, NaN. Sve ostalo je truthy.

  7. For the following code, what will be logged to console and why.

    1. console.log( 4 == "4" && true == 1)

      Answer: Točno. Oba uvjeta su istinita.

    2. console.log( true && null )

      Answer: Null. && vraća prvu falsy vrijednost.

    3. console.log("House" && "Mouse")

      Answer: 'Mouse'. Obje vrijednosti su truthy, vraća drugu.

    4. console.log(false && "Mouse")

      Answer: Netočno. Vraća se prva falsy vrijednost.

    5. console.log("Mouse" && false)

      Answer: Točno. Vraća prvu falsy vrijednost.

    6. console.log(0 && false)

      Answer: 0 — falsy.

    7. console.log( 4 == "4" || true == 1)

      Answer: Točno. Jedan od uvjeta je istinit.

    8. console.log(true || null)

      Answer: Točno. Operator || vraća prvu truthy vrijednost.

    9. console.log("House" || "Mouse")

      Answer: House', vraća prvu truthy vrijednost.

    10. console.log(false || "Mouse")

      Answer: 'Mouse' ,vraća prvu truthy vrijednost.

    11. console.log("Mouse" || false)

      Answer: 'Mouse' vraća prvu truthy vrijednost.

    12. console.log(0 || false)

      Answer: Netočno, oboje falsy.

    13. console.log(true || false && false;) hint

      Answer: Točno. && se evaluira prvo → false && false → false, zatim true || false → true.

    14. console.log((true || false) && false;)

      Answer: Netočno, zagrade se evaluiraju prve → (true || false) = true → true && false = false.
